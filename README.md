# 📝 Lista de Tareas JS

Aplicación web para agregar y eliminar tareas, desarrollada con HTML, CSS y JavaScript.

## 🚀 Demo
[Ver proyecto en línea](https://lista-tareas-angel.netlify.app/)

## 🛠️ Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- LocalStorage

## ✨ Funcionalidades
- Agregar tareas
- Eliminar tareas
- Guardar tareas en el navegador
- Mantener tareas al recargar la página

## 📂 Estructura del proyecto
- `index.html`
- `style.css`
- `script.js`

## ▶️ Cómo ejecutar el proyecto
1. Clona este repositorio
2. Abre el archivo `index.html` en tu navegador
